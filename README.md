Webber - Handy base image for webtesting
========================================

This is a base image based on Ubuntu, with installed:

- Python
- Pip
- Virtualenv 
- PhantomJS
- Git
- postgresql-common
- xvfb
- firefox

It's handy if you want to run beahvioural tests via Selenium, using a PhantomJS driver
or a Firefox vua `xvfb-run`
